Как пользоваться SmartGrid:
1 Заполни config для сетки в gulpfile.js
```const smartGridConf = {
    outputStyle: 'scss',
    columns: 12,
    offset: "30px",
    mobileFirst: false,
    container: {
        maxWidth: "1200px",
        fields: "30px"
    },
    breakPoints: {
        lg: {
            width: "1200px",
            fields: "30px"
        },
        md: {
            width: "992px",
            fields: "20px"
        },
        sm: {
            width: "768px",
            fields: "15px"
        },
        xs: {
            width: "576px",
            fields: "10px"
        },
        xxs: {
            width: "480px",
            fields: "5px"
        }
    }
};

gulp.task('grid', () => {
    smartgrid('src/css/scss', smartGridConf);
});```
(edited)

2 Сгенерируй файл смарт грида
*gulp grid*

3 К контейнеру где нужно переставлять элементы пишем:
```.row-flex();
//и свойство распределения элементов в файле, например
justify-content: space-between;```

4 к элементам внутри контейнера
```.class_элемента {
        .col(); // говорим у нам сетка на марджинах
        .size(2); // этот элемент(или элементы) на самой большой ширине занимает 2 колонки
        .size-md(12); // этот элемент(или элементы) на ширине md занимает 12 колонок
}```
(edited)

5 в случае если нужно поменять набор свойств делаем так:
```.class_элемента {
        .md-block({ // на ширине md применить данные свойства к этому элементу
          display: block;
          text-align: center;
          margin-top: 0;
        });
        .sm-block({ // на ширине sm применить данные свойства к этому элементу
          width: 0;
          margin: 0 auto;
          text-align: center;
          display: inline;
        });
}```
***еще перед началом нужно использовать .reset(); // сброс настроек браузера для css
***и к ширине сайта можно применить  .wrapper(); // создаем центрующий блок
ну в принципе ты это сделал
и установи плагины:
*gulp-concat* // объединит файлы js и css(less) в один
*gulp-group-css-media-queries* // объединит все одноширинные запросы(media-queries) что бы не создавать кучу все вопхнет в один media-queries
*gulp-sourcemaps* // укажет путь к строке твоего less
*gulp-imagemin* // оптимизирует jpeg png файлы для браузеров
*gulp-clean-css* // очистит css от пробелов комментов и пр. в общем ужмет
*gulp-htmlmin* // минификация html
*gulp-rename* // удобно создавать минифайлы типа: main.min.js
*gulp-babel* // преобразует es6 в es5 (edited)

еще если нужно в смарт грид отступить одно или несколько колонок, делаем так:
.shift-left — Отступ слева
.shift-md-right — Отступ справа на md ширине
